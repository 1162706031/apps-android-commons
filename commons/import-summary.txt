Moved Files:
------------
Android Gradle projects use a different directory structure than ADT
Eclipse projects. Here's how the projects were restructured:

* AndroidManifest.xml => app/src/main/AndroidManifest.xml
* assets/ => app/src/main/assets/
* jni/ => app/src/main/jni/
* res/ => app/src/main/res/
* src/ => app/src/main/java/
* src/main/native/Android.mk => app/src/main/resources/main/native/Android.mk
* src/main/native/README.gst-sdk => app/src/main/resources/main/native/README.gst-sdk
* src/main/native/gst-build/gstreamer_android.c => app/src/main/resources/main/native/gst-build/gstreamer_android.c
* src/main/native/gst-build/gstreamer_android.o => app/src/main/resources/main/native/gst-build/gstreamer_android.o
* src/main/native/transcode.c => app/src/main/resources/main/native/transcode.c
